# Requisitos del Proyecto de la App de Trivia

## Descripción general
Una aplicación web de trivia donde los usuarios pueden responder preguntas de varias categorías, seguir sus puntajes y gestionar sus perfiles. El banco de preguntas está pensado para generarse mediante IA.

## Funcionalidades principales
1.  **Banco de preguntas generado por IA:** Sistema para generar y almacenar preguntas/respuestas.
2.  **Integración con base de datos:** Almacenamiento de preguntas, usuarios y puntajes.
3.  **API CRUD:** Endpoints de backend para gestionar:
    *   Preguntas y respuestas
    *   Perfiles de usuario
    *   Puntajes de usuarios
4.  **Autenticación:** Inicio de sesión seguro para usuarios.
5.  **Puntuación de usuarios:** Seguimiento y visualización de puntos por respuestas correctas.

## Pantallas propuestas para diseño
1.  **Página de inicio de sesión/registro:** Punto de entrada simple y seguro.
2.  **Dashboard/Inicio:** Vista general de categorías, puntajes recientes y un botón de "Iniciar quiz".
3.  **Pantalla de quiz:** La experiencia central donde se presentan preguntas con respuestas de opción múltiple.
4.  **Pantalla de resultados:** Retroalimentación del desempeño, mostrando puntaje final y respuestas correctas/incorrectas.
5.  **Panel de administración (CRUD):** Interfaz de gestión para que administradores creen, lean, actualicen y eliminen preguntas manualmente.
6.  **Ranking/Perfil:** Muestra el posicionamiento de usuarios y estadísticas personales.
