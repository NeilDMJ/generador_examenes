# Quizzly Pulse

## Descripcion de la pagina

Esta aplicacion corresponde a **Quizzly Pulse**, una plataforma web de trivia y generacion de examenes con enfoque academico. Basada en la propuesta del proyecto mencionada en clase, la pagina incluye autenticacion de usuarios, dashboard con categorias, flujo de quiz por preguntas, resultados con puntajes, ranking/perfil y un panel administrativo CRUD para gestionar preguntas, respuestas, usuarios y puntuaciones. Ademas, contempla la integracion de un banco de preguntas generado con IA.
